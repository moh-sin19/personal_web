## Getting started

**1. Install dependencies**
```
npm install
```

**2. Run dev server**
```
npm run dev
```

**3. Edit config**
```
src/resources/once-ui.config.js
```

**4. Edit content**
```
src/resources/content.js
```

**5. Create blog posts / projects**
```
Add a new .mdx file to src/app/blog/posts or src/app/work/projects
```
